# Cloud Deploy

Cloud Deploy is a lightweight web tool for deploying Lyquid projects.

The tool lets a user import a Lyquid ABI, configure the target RPC endpoint, upload a Lyquid working directory, run the ABI-selected build flow, review the prepared deployment output, connect a wallet, and deploy or update a Lyquid.

The project is intentionally small. It is a deployment utility, not a marketplace, registry, provenance system, or general Lyquid management platform.
